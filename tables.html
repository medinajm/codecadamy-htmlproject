HTML Reference Page

HTML Tables

Tag Elements
Tag Name Description
<table> Table Starts a new table inside the html document
<tr> Table Row Creates a new row for your table
<td> Table Data Adds data to the cell inside your table
<th> Table Heading Adds a title to your rows and columns and must be enclosed in a <tr> element
<thead> Table Head This element defines the table headings encapsulated in tabel rows
<tbody> Table Body separates the body section from the header and footer of the table
<tfoot> Table Footer Gives your table a footer section


Attributes
Attribute Name Description
colspan Column Span Defines how many columns this cell should take up
rowspan Row Span Defines how many rows this cell should take up
